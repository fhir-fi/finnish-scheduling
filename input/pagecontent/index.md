This implementation guide specifies the Finnish FHIR profiles related to scheduling.

It builds on two previous implementation guides published by HL7 Finland:
* [Finnish Scheduling IG](https://simplifier.net/finnishschedulingr4/) published in 2020
* [Finnish Appointment IG](https://simplifier.net/finnishappointment) published in 2021

It also draws inspiration from other sources, particularly from:
* [Appointment Scheduling Workflows](https://www.hl7.org/fhir/appointment.html#workflow-types) in
  the FHIR specification
* [Argonaut Scheduling](https://fhir.org/guides/argonaut/scheduling/) implementation guide
* [Scheduling Building Blocks](https://confluence.hl7.org/display/AP/Scheduling+Building+Blocks)
* [SMART Scheduling Links](https://github.com/smart-on-fhir/smart-scheduling-links)

See also the implementation guide for
[Finnish Base Profiles](https://hl7.fi/fhir/finnish-base-profiles/).

All implementation guides published by HL7 Finland can be found at
[hl7.fi/fhir](https://hl7.fi/fhir/).
